## 测试环境要求

### 客户端环境
- 操作系统：Windows 10/11, macOS
- 浏览器：Chrome 最新版, Firefox 最新版, Safari 最新版
- 屏幕分辨率：最低 1366 x 768

### 服务端环境
- Node.js >= 14.x
- MySQL >= 5.7
- Redis >= 6.x
